# Akan Name Generator

## 📖 Project Description
This is a simple web application that calculates the day of the week a user was born and assigns them an Akan name based on their gender.
Akan names originate from Ghanaian culture, where children are named according to the day of the week they were born.

## 🧪 Behavior Driven Development (BDD)

| Input | Action | Output |
|------|--------|--------|
| Valid date + gender | Click button | Displays correct Akan name |
| Invalid day (e.g. 32) | Click button | Alert "Invalid day" |
| Invalid month (e.g. 13) | Click button | Alert "Invalid month" |
| No gender selected | Click button | Alert "Select gender" |
| Empty name field | Click button | Alert "Enter your name" |
