# Akan Name Generator

##  Project Description
This is a simple web application that calculates the day of the week a user was born and assigns them an Akan name based on their gender.
Akan names originate from Ghanaian culture, where children are named according to the day of the week they were born.

##  Setup Instructions

1. Clone or download the project folder
2. Open the folder in your code editor (e.g., VS Code)
3. Ensure the following files are present:
   - index.html
   - styles.css
   - script.js
4. Open index.html in your browser

##  Behavior Driven Development (BDD)

| Input | Action | Output |
|------|--------|--------|
| Valid date + gender + name | Click button | Displays correct Akan name |
|No date |click button |alert "please fill in this field" |
|No month | click button |alert "please fill in this field" |
| No year | click button |alert "please fill in this field" |
| Empty name field | Click button |alert "please fill in this field" |
